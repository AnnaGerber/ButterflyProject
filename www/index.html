Butterfly Project

Exploring the Fundamental Interconnectedness of All Things

About

In the Butterfly Project, we explore the the butterfly effect, described by Edward Lorenz, where small changes in one place, such as the flapping of the wings of a butterfly, can result in large changes, like a hurricane, elsewhere. Or, as Douglas Adams puts it:

what we are concerned with here is the fundamental interconnectedness of all things ... The connections between causes and effects are often much more subtle and complex than we with our rough and ready understanding of the physical world might naturally suppose Douglas Adams, Dirk Gently's Holistic Detective Agency

This project utilizes a global network of sensor data published through services such as cosm.com, contributing local sensor data, and aggregating sensor data from around the world, with the aim of enabling people to discover and explore correlations between seemingly unrelated sensor data streams.

As the internet of things expands and the amount of sensor data available online increases, data visualisation tools such as these will contribute to our improved understanding of the nature of the fundamental interconnectedness of all things.

Implementation

Sensor unit: the metaphorical butterfly

  • An Arduino nano microcontroller polls the DHT11 temperature and humidity sensor periodically to obtain readings
  • The Arduino publishes the sensor values as CSV via Serial
  • Implemented using Arduino wiring language

Publication of the sensor stream

  • The publishing module listens via Serial, parses and publishes the sensor values to COSM.
  • Implemented in Java

Visualisation

  • The visualisation service retrieves sensor values for this stream and others around the world using the COSM API.
  • Implemented in PHP and HTML5
  • Deployed on the NeCTAR research cloud
